# Campus Explorer - Interactive Room Query System

**Course Project**: CPSC 310 - Introduction to Software Engineering (UBC)  

[![Demo Video](https://img.youtube.com/vi/GLBa6bYwOJw/maxresdefault.jpg)](https://www.youtube.com/watch?v=GLBa6bYwOJw)

## Overview

Campus Explorer is a full-stack web application that enables real-time querying and visualization of 300+ UBC campus rooms through an interactive interface powered by Google Maps API.

## Tech Stack

**Backend:**
- TypeScript
- Node.js
- Express.js

**Frontend:**
- JavaScript
- HTML/CSS
- Google Maps API

## Key Features

- **RESTful API** with comprehensive dataset management (add/remove/query operations)
- **Complex Query Parser** supporting filters, aggregations (MAX/MIN/AVG/COUNT/SUM), grouping, and multi-key sorting
- **HTML Parser** (parse5) for processing building/room data from HTML tables
- **Geolocation Integration** via external HTTP API
- **Interactive Map Visualization** with custom markers and info windows
- **Real-time Search** with autocomplete functionality

## Project Highlights

- Processed and managed datasets containing 300+ campus rooms
- Achieved 100% data extraction accuracy from HTML documents
- Implemented robust error handling and validation throughout the application
- Designed and built both frontend and backend from scratch following software engineering best practices
